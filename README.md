# [A Hands-on Workshop on AI and Data Science for Advancing Translational Science](https://apexbtic.icgeb.res.in/workshop2025/)

## [Day-3 Hands-on Session: Introduction to Machine learning for Tabular Data](https://apexbtic.icgeb.res.in/workshop2025/lecture/ishaan.html)

### Introduction
This hands-on session will briefly introduce key steps in a typical machine learning workflow to analyze tabular data using models like SVM, where we will handle missing values, class imbalance, and data scaling, followed by model training, evaluation, and interpretation. Additionally, we will explore a tabular foundational deep learning model to analyze the same data and compare its performance and interpretability with classical approaches.

### Datasets
1. [ST000385](https://www.metabolomicsworkbench.org/data/DRCCMetadata.php?Mode=Study&StudyType=&StudyID=ST000385)

Requirement:
> Git, Python, Jupyter Notebook, Pandas, Numpy, Scikit-Learn, Imbalanced-learn, Matplotlib, SHAP

Clone the repo:
```

$ cd ML-for-Tabular-Data-ICGEB-Workshop-2025/
```
Run Jupyter Notebook

- Select the notebook (ml_analysis, dl_analysis)
